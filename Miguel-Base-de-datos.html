La Base de Datos:

Una base de datos sólidamente estructurada constituye el cimiento robusto sobre el cual descansa la eficiencia y confiabilidad de cualquier sistema informático.

Esta base de datos fue diseñada no solo para enfretar los problemas actuales de la gestión, se diseñó con la posibilidad de explotar al máximo el análisis de los datos almacenados, a fin de no limitar la busqueda de ventajas competitivas.

Además que su estructura permite la facil instegración de nuevas características a fin de acompañar el crecimiento de la empresa. Contando así con un diseño previsor a fín de poder adaptarse a requisitos nuevos.

Posibilidades Previstas de la Base de datos

  • Llevar el registro de las actividades comerciales de la empresa.
  • Realizar el seguimiento al Stock
  • Gestión de alertas en lo relativo a Stock, Plazos entre reclamos y rendimientos comerciales.
  • Facil integración de nuevas caracterististicas.
  • Servir como apoyo para la elaboración de los registros contables
  • Registro y seguimiento de los reclamos de productos
  • Análisis de las transacciones clasificadas por: rubros, productos, clientes, proveedores, región, fechas, formas de pago, entre otros.
